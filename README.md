 Innovael Dashboard

A modern, responsive dashboard template built with HTML, CSS, and Bootstrap 5. Designed for clean, functional web applications and admin panels.  

Live Demo: [https://innovael-dashboard.vercel.app](https://innovael-dashboard.vercel.app)

Features:

- Fully responsive layout for desktop and mobile  
- Organized folder structure for maintainability  
- Built with Bootstrap 5 for rapid and consistent styling  
- Ready-to-use template for dashboards and admin panels  

Project Structure
Innovael-Dashboard/
├── html/
│ └── index.html # Main dashboard page
├── css/
│ └── style.css # Custom styles
└── js/
└── bootstrap.bundle.js # Bootstrap JS components
The folder structure keeps code organized and easy to maintain.

Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- Bootstrap 5
